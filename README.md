# Administração do Catálogo de Videos

## Entidade Categoria

### Campos
- Nome
- Descrição
- Ativar ou desativar
- Auditoria (data de criação, atualização e deleção)

### Regras de Negócio
- Nome não deve ser nulo ou em branco
- Nome deve conter no mínimo 3 caracteres
- Descrição pode ser nula ou em branco
